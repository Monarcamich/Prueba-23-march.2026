# F1 Rhythm Analysis: Pace Progression in Formula 1 Racing

## Overview

A comprehensive Python project analyzing pace progression patterns in Formula 1 races. This project investigates:

- **Lap Time Progression**: How lap times change throughout a race
- **Driver Rhythm Comparison**: Comparing pace between different drivers
- **Tire Degradation**: Estimating tire wear effects on lap times
- **Strategic Insights**: Data-driven analysis of race strategies and performance

## Core Concepts

### ETL Architecture (3-Layer Pattern)

#### Bronze Layer
- **Purpose**: Store raw data as-is from sources
- **Format**: JSON files
- **Files**: `data/bronze/{season}_r{round}_raw.json`

#### Silver Layer  
- **Purpose**: Clean, validate, and normalize data
- **Operations**:
  - Convert lap times MM:SS.mmm → seconds (float)
  - Remove invalid records
  - Standardize data types
- **Files**: `data/silver/{season}_r{round}_clean.csv`

#### Gold Layer
- **Purpose**: Create analysis-ready datasets with derived metrics
- **Adds**:
  - `pace_delta`: Lap-to-lap time change (seconds)
  - `lap_position`: Driver's rank by time in each lap
  - `degradation_rate`: Tire wear rate (seconds/lap)
- **Files**: `data/gold/{season}_r{round}_analysis.csv`

### Data Dictionary

#### Core Columns (Silver/Gold)
| Column | Type | Description |
|--------|------|-------------|
| season | int | F1 season year |
| round | int | Race round number |
| lap_num | int | Lap number in race |
| driver_id | str | Unique driver identifier |
| lap_time_seconds | float | Lap time in seconds |

#### Derived Metrics (Gold)
| Column | Type | Description |
|--------|------|-------------|
| pace_delta | float | Change from previous lap (seconds) |
| lap_position | int | Driver's rank by time in lap |
| degradation_rate | float | Tire wear rate (seconds/lap) |

## Usage Examples

### Load and Analyze a Single Race
```python
from src.data_fetcher import get_fetcher
from src.etl_pipeline import Pipeline
import pandas as pd

# Fetch data
fetcher = get_fetcher(use_mock=True)
season_data = fetcher.fetch_season(2024)
laps_data = fetcher.fetch_race_laps(2024, 1)

# Process through pipeline
pipeline = Pipeline()
race_data = {**season_data['races'][0], **laps_data}
analysis_df = pipeline.process_race(2024, 1, race_data)

# Analyze
print(f"Drivers: {analysis_df['driver_id'].nunique()}")
print(f"Laps: {analysis_df['lap_num'].max()}")
print("\nTop 5 by pace:")
top_drivers = analysis_df.groupby('driver_id')['lap_time_seconds'].mean().sort_values().head(5)
print(top_drivers)
```

### Compare Degradation Rates
```python
# Which drivers degrade faster?
degradation = analysis_df[['driver_id', 'degradation_rate']].drop_duplicates()
degradation = degradation.sort_values('degradation_rate', ascending=False)
print(degradation)
```

### Analyze Pace Evolution
```python
# How does lap pace change as tire degrades?
driver = 'max_verstappen'
driver_data = analysis_df[analysis_df['driver_id'] == driver].sort_values('lap_num')

import matplotlib.pyplot as plt
plt.figure(figsize=(10, 6))
plt.plot(driver_data['lap_num'], driver_data['lap_time_seconds'], marker='o')
plt.xlabel('Lap Number')
plt.ylabel('Lap Time (seconds)')
plt.title(f'{driver} - Pace Progression')
plt.grid(True)
plt.show()
```
